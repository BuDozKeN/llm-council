# AI Council Knowledge Base

This folder contains exported conversations from the AI Council for reference and article writing.

## Export Format

Each exported file includes:
- **Question** - The original question asked to the council
- **AI Council Answer** - The synthesized final answer from the council chairman
- **Individual Model Responses** (collapsible) - Responses from each council member
- **Peer Rankings** (collapsible) - How each model ranked the responses
